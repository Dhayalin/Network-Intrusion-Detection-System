# Network-Intrusion-Detection-System
A real-time Network Intrusion Detection System built with Python and Flask. The system captures raw network packets using low-level socket programming, parses them across the Ethernet, IP, and transport layers, and feeds them through a stateful rule engine that detects threats like port scans, SYN floods, ICMP sweeps, and brute-force attacks using sliding time-window analysis. Detected threats are classified by severity (Critical / High / Medium / Low), logged to disk in JSON Lines format, and surfaced through an interactive web dashboard that updates live every 2 seconds with packet feeds, traffic charts, and an alert feed — no page refresh needed.
